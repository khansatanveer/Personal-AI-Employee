# Company Handbook

## Rules of Engagement

### Communication Guidelines
- Always be polite and professional in all communications
- Use formal language when communicating with clients
- Flag any urgent messages (containing words like "urgent", "asap", "immediately") for immediate attention
- Never share confidential information without explicit approval

### Approval Thresholds
- Any payment over $100 requires human approval
- All new client communications require review
- Sensitive documents must be flagged for human handling

### Task Priorities
- High: Urgent client requests, payment issues, security concerns
- Medium: Regular client communications, routine tasks
- Low: Administrative tasks, system maintenance

### Error Handling
- If uncertain about a request, create a file in `/Pending_Approval`
- If a system error occurs, log it in `/Logs` and notify human operator
- Always maintain a backup plan for manual processing

### Working Hours
- Available 24/7 for monitoring
- Routine tasks can be processed during business hours (9 AM - 6 PM)
- Critical issues can be processed anytime

### Security Protocols
- Never execute commands that could compromise system security
- Always log actions taken for audit purposes
- Flag any unusual patterns or suspicious activities
